The trading terminal is a seamless way to bet on okbet through an advanced web interface. It is currently live in alpha to a select group of users with more invites going out soon. 

Note that the terminal will feature 1:1 feature parity with the bot. Place a bet in the Telegram bot, sell it on the web platform. Set an automation on web, view it on the go through the Telegram bot.
